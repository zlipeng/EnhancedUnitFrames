# Enhanced Unit Frames UI 设计规范

> **设计目标**：为魔兽世界12.0版本单位框体增强插件设计完整UI系统，包括设置面板、编辑模式交互和最终视觉效果展示。

---

## 一、设计原则

### 1.1 核心设计理念

| 原则 | 说明 | 12.0 合规关联 |
|------|------|---------------|
| **暴雪原生风格** | 延续WoW官方UI视觉语言，降低用户学习成本 | 与游戏原生UI融合，避免视觉冲突 |
| **渐进式披露** | 常用功能前置，高级选项折叠隐藏 | 减少战斗中误操作风险 |
| **即时预览** | 所有修改实时反映到游戏界面 | 非战斗时即时生效，战斗中明确提示 |
| **安全优先** | 危险操作需二次确认，状态清晰可见 | 符合Secret Value和Secure Frame限制 |

## 三、编辑模式交互设计

### 3.1 编辑模式进入流程

```
用户输入 /editmode
        │
        ▼
┌───────────────────────────────┐
│  游戏进入编辑模式              │
│  ├── 所有可编辑UI高亮显示      │

        │
        ▼
┌───────────────────────────────┐

│  ├── 蓝色高亮边框              │

        │
        ▼
┌───────────────────────────────┐

```
